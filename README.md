# UsingBus
###应用简介
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;快用公交是一款基于高德SDK开发的公交车查询软件，可以根据用户指定的出发和到达地点搜索公交换乘方案，并根据最少时间，最少换乘，最少步行提供最优化的换乘方案，也可以根据用户的搜索要求搜索用户周围线路，站点等， 当然也可以搜索指定线路公交的途径站点，运营时间，票价等等。
